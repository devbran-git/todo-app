## Este é o to.do app
Aplicação baseada na trilha React Native do bootcamp Ignite da Rocketseat.

<br>

**Funcionalidades:**
- Inserir/Alterar/Deletar tarefas;
- Marcar como feito;
- Modal de alerta de confirmação par deletar a tarefa.
